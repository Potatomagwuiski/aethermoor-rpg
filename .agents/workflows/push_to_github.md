description: Push changes to GitHub
---

Follow these steps to systematically commit and push all recent changes to GitHub.

1. First, check the status of the repository and identify what files were modified to help you form a good commit message.
// turbo
`git status`

2. Stage all modifications and new files.
// turbo
`git add .`

3. Generate a concise, descriptive commit message summarizing the work you just completed, and commit the changes. (Replace 'Your commit message here' with the actual description)
`git commit -m "Your commit message here"`

4. Push the changes to the remote repository.
// turbo
`git push`
